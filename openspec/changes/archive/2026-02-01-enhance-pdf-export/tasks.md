  - JSON 文件存储
  - 记录字段: task_id, style, title, timestamp, pdf_file
  - 限制最多 1000 条记录
  - 验证：设计文档完成

- [x] **T2.4.2** 实现历史记录功能
  - `_add_export_history()` 记录函数
