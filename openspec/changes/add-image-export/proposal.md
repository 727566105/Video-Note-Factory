# 导出图文功能

## 概述

新增"导出图文"功能，将生成的笔记内容渲染为精美的图片格式（类似小红书分享卡片），方便用户在社交媒体平台分享。

## 背景与动机

当前系统已支持 PDF 导出，但 PDF 格式不适合在社交媒体分享。用户希望有一种更直观、更具视觉吸引力的方式来分享笔记内容，特别是对于"小红书"风格的笔记，需要能够生成类似社交媒体分享卡片的图片格式。

**用户需求：**
- 将笔记内容渲染为一张或多张图片
- 图片样式类似小红书的分享和安利笔记图文
- 方便在微信、朋友圈、小红书等平台分享
- 支持多种图文模板和样式

## 目标

1. 提供图文导出 API 端点，支持将笔记转换为图片
2. 创建多种图文模板（小红书风格、简约风格、学术风格等）
3. 前端新增"导出图文"按钮，与现有"导出 PDF"按钮并列
4. 支持自定义图文样式（背景、字体、布局等）

## 范围

### 包含
- 后端图文渲染 API（使用 html2canvas 或 puppeteer）
- 前端导出按钮和用户交互
- 多种预设图文模板
- 图文预览功能
- 导出历史记录

### 不包含
- 视频内容截图（已有的 video_understanding 功能）
- GIF 动图生成
- 批量图文导出（可在后续版本添加）

## 依赖

- 后端：使用 html2canvas（Python）或 Playwright 进行 HTML 到图片的转换
- 前端：现有的 Markdown 渲染组件
- 模板设计：需要 UI/UX 设计师提供模板规范
