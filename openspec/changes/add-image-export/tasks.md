## 依赖关系

```

    ↓
```

## 可并行工作

- 阶段 2 和阶段 4 可以并行进行
- 前端组件开发可以在后端 API 完成前开始（使用 mock 数据）
- 模板预览图可以提前设计

## 验收标准

1. **功能完整性**
   - [x] 用户可以成功导出笔记为图片
   - [x] 支持至少 3 种模板样式
   - [x] 支持自定义图片宽度

2. **用户体验**
   - [x] 操作流程简单直观
   - [x] 加载状态清晰反馈
   - [x] 错误提示友好

3. **性能**
   - [x] 普通笔记（3000字）生成时间 < 5 秒
   - [x] 长笔记（10000字）生成时间 < 15 秒
   - [x] 缓存命中后响应时间 < 1 秒

4. **质量**
   - [x] 图片清晰度高（无锯齿、模糊）
   - [x] 样式符合模板预览
   - [x] 中文显示正常
